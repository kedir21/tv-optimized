# CineStream TV

A premium, cinematic Android TV-style streaming application using TMDB metadata and iframe-based playback. Optimized for remote control navigation and large screens.

## Features

- **TV-First Design**: Large touch targets, focus states, and spatial navigation support.
- **TMDB Integration**: Real-time trending, popular, and top-rated movies/shows.
- **Watchlist**: Save your favorite content locally.
- **Multi-Source Player**: Supports VidSrc and RiveStream embeds.
- **Responsive**: Adapts from mobile phones to 4K TVs.

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run development server**:
    ```bash
    npm run dev
    ```

3.  **Build for production**:
    ```bash
    npm run build
    ```

## Deployment

This project is configured for **Vercel** or **Netlify**. Simply push this repository to GitHub and connect it to your preferred hosting provider.
